# Turtle soccer game
- 기간 : 2026.5.19~2026.5.21
- 기술 : `Python` `Turtle`
- 설명 : Turtle 모듈을 이용한 게임 만들기


## 한계 및 보완점
- **현상:**
    - 게임 플레이 중 공이 골대 안으로 들어갔음에도 판정이 모호하게 처리되거나, 골키퍼의 수비 모션과 실제 ‘골/막힘’의 기준이 일치하지 않는 현상이 발생함.
    
- **원인 분석:**
    1. 그래픽 이미지의 시각적 형태(예: 가랑이 사이의 빈 공간 등)와 컴퓨터가 계산하는 캐릭터의 중심 좌표 사이에 오차가 존재하여 충돌 판정의 정확도가 떨어짐.
    2. 축구공의 이동 속도가 빠르다 보니, 0.01초 단위로 위치를 계산하는 과정에서 가랑이 사이의 좁은 판정 범위를 순식간에 통과해 버리는 타이밍 오류가 발생함.

- **보완 방향:**
    - 이미지 크기에 매칭되는 정밀한 픽셀 단위 범위를 재정립하고, 예외적인 상황(가랑이 사이 골 등)을 처리할 수 있는 다중 조건문을 도입하여 판정의 정확도를 높여야 함.
